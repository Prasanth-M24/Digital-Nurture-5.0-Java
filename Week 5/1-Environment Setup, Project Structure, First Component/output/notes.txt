Task 1: Scaffold and Explore the Angular Project - Expected outcomes
Explanations for the files in the Angular Project
    1. angular.json: Conatins Angular project configuration and build settings
    2. tsconfig.json: It has the main typescript compiler configuration
    3. tsconfig.app.json: Typescript settings for application
    4. package.json: Stores the project information and dependencies
    5. main.ts: Entry point that starts the Angular application
    6. app.config.ts: Stores application-wide configuration and providers (Standalone Angular)
    7. app.component.ts: Defines the root component logic
    8. index.html: Main HTML page where Angular loads.
